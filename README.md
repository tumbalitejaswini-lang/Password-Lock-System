# Password-Lock-System
Built a Prototype model of a Lock system using Push buttons . Here push buttons assumed to be  numbers , where as  password can be set before hand in the programming logic .

~ PROBLEM STATEMENT :
* In everyday security systems like door locks, lockers, and access panels, password authentication is one of the most common methods used to prevent unauthorized access. However, most advanced systems are costly and complex for beginners to understand.
* This project was developed as a simple and affordable way to demonstrate how a basic password-based security system works using embedded components.

~ OBJECTIVE :
To design a functional password authentication system that helps understand:
* Digital input handling using push buttons
* Real-time decision making in embedded systems
* Output control using LEDs and buzzer
* Core logic behind security-based systems

~ PROJECT DESCRIPTION :
* This system uses an Arduino Uno as the controller and push buttons as password inputs.
* The user enters a predefined 4-step password using buttons. The system stores the input sequence and compares it with the correct password.
* Based on validation:
🟢 Correct password → Green LED + success beep (Access Granted)
🔴 Wrong password → Red LED + alarm sound (Access Denied)
(The system resets automatically after each attempt, making it reusable and interactive.)

~ WORKING FLOW :
 [ Input → Validate → Output → Reset ]
* User enters password via push buttons
* Arduino records input sequence
* After 4 inputs, system validates password
* Correct → Green LED + beep
* Wrong → Red LED + alarm
* System resets for next attempt

~ FEATURES :
* Security and reliability
* Easy to access
* Simple construction
* Flexible model
